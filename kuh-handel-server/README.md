# kuh-handel-server

WebSocket game server for the [Kuh-Handel](https://ufuk-guenes.com/kuh-handel) online bot coding challenge. Handles player registration, turn-taking, game state, and result tracking.

The server is part of the [kuh-handel](https://github.com/ufukguenes/kuh-handel) mono-repo.

## Participating

If you want to build a bot and compete, head to the [website](https://ufuk-guenes.com/kuh-handel) for documentation and rules. Bot clients are available as:

- Rust: [kuh-handel-lib](https://crates.io/crates/kuh-handel-lib)
- Python: [pyhandel](https://pypi.org/project/pyhandel/)
